# 電力生怪磚
黏液科技的附加 繁體翻譯版

可讓你的損壞生怪磚變成電力生怪磚.<br>
電力生怪磚與普通的生怪磚運作方式相似,但他是使用電力來生怪,而且也不需要玩家在附近.

> 此為**非官方**版本, 請勿在該作者問題追蹤內回報! <br>
